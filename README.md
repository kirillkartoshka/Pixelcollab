# PixelCollab 🎨

Совместное пиксельное рисование в реальном времени.  
Открывайте холст, выбирайте цвет и творите вместе с друзьями прямо в браузере.

## ✨ Возможности
- 🎨 Палитра из 17 цветов
- 🔲 Регулируемый размер пикселя (4–20px)
- 🔍 Масштабирование колёсиком мыши или кнопками
- 💾 Сохранение рисунка в PNG
- 👥 Счётчик онлайн‑пользователей
- ⚡️ Мгновенная синхронизация через Firebase

## 🚀 Быстрый старт
1. Склонируйте репозиторий
2. В файле index.html замените firebaseConfig своими данными из Firebase Console
3. Включите Realtime Database в проекте Firebase
4. Откройте index.html в браузере — можно рисовать!
